# Blueprint IQ

AI-powered blueprint reading quiz tool designed to train construction professionals and reduce onboarding errors in the field.

> **Status:** In active development

## What It Does

Blueprint IQ generates adaptive quizzes from construction blueprints using AI, helping tradespeople and new hires rapidly build blueprint literacy. Questions are dynamically generated based on uploaded blueprint images, with difficulty that adjusts to the user's performance.

## Tech Stack

- **Frontend:** React, Vite
- **AI:** Anthropic Claude API (prompt engineering for domain-specific question generation)
- **Styling:** CSS

## Features (In Development)

- AI-generated quiz questions from blueprint images
- Adaptive difficulty engine
- Score tracking
- Construction-domain focused output quality

## Background

Built from a real gap identified through hands-on experience with a family-owned general contracting business. Blueprint misreading is a leading cause of errors and rework on job sites — this tool targets that problem directly.
